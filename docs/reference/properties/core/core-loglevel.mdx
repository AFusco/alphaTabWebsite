## Description
AlphaTab internally does quite a bit of logging for debugging and informational purposes. The log level of alphaTab can be controlled via this setting.

<PropertyDescription metadata={metadata} />

## Types

<TypeTable>
    <TypeRow type="net" name="AlphaTab.Util.LogLevel">
        `None` -  No logging
        `Debug` - Debug level (internal details are displayed)
        `Info` - Info level (only important details are shown)
        `Warning`
        `Error`
    </TypeRow>
    <TypeRow type="json-html" name="string">
        `none`
        `debug`
        `info`
        `warning`
        `error`
    </TypeRow>
    <TypeRow type="js-html" name="int">
        `0` - None
        `1` - Debug
        `2` - Info
        `3` - Warning
        `4` - Error
    </TypeRow>
</TypeTable>

## Default Value

`Info`
