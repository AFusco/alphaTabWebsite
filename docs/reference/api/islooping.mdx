## Description
This setting controls whether the playback should automatically restart after it finished to create a playback loop.

## Types

<TypeTable>
    <TypeRow type="all" name="bool" />
</TypeTable>

## Default Value

`false`

## Example - C#

```csharp
var api = new AlphaTabApi<MyControl>(...);
api.IsLooping = true;
```

## Example - JavaScript

```js
var api = new alphaTab.AlphaTabApi(document.querySelector('#alphaTab'));
api.isLooping = true;
```

## Example - jQuery

```js
$('#alphaTab').alphaTab('looping', true)
```
