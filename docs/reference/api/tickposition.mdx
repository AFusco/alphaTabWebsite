## Description
Gets or sets the player position in midi ticks. 

## Types

<TypeTable>
    <TypeRow type="all" name="int" />
</TypeTable>

## Example - C#

```csharp
var api = new AlphaTabApi<MyControl>(...);
api.TickPosition = 4000;
```

## Example - JavaScript

```js
var api = new alphaTab.AlphaTabApi(document.querySelector('#alphaTab'));
api.tickPosition = 4000;
```

## Example - jQuery

```js
$('#alphaTab').alphaTab('tickPosition', 4000)
```
