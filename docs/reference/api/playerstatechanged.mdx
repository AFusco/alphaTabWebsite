## Description
This event is fired when the playback state changed.

## AlphaTab.Audio.Synth.PlayerStateChangedEventArgs Properties

<ParameterTable>
    <ParameterRow platform="net" name="State" type="AlphaTab.Audio.Synth.PlayerState">
        The new state of the player.  
    </ParameterRow>
    <ParameterRow platform="js" name="state" type="int">
        The new state of the player. <br />
        `0` - Paused
        `1` - Playing
    </ParameterRow>
    <ParameterRow platform="js" name="stopped" type="bool">
        Whether the playback was fully stopped (including jump to start) or only paused.
    </ParameterRow>
</ParameterTable>

```js
    const args = e.detail;
    updatePlayerControls(args.state, args.stopped);
}, false);
```
