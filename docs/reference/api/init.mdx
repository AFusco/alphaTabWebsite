## Description
This jQuery specific function initializes alphaTab. For other systems this is not needed as the API is directly created. 

## Signatures

<TypeTable>
    <TypeRow type="jquery" name="alphaTab(settings)" />
</TypeTable>

### Parameters

<ParameterTable>
    <ParameterRow platform="jquery" name="settings" type="AlphaTab.Settings">
        The settings to use for alphaTab. Either as JSON structure or as Settings instance.
    </ParameterRow>
</ParameterTable>

### Returns
Nothing

## Example - jQuery

```js
$('#alphaTab').alphaTab({
    scale: 1.5
});
```
