## Description
Gets access to the underling <code>AlphaTab.Settings</code> object that is currently used by AlphaTab. 

## Types

<TypeTable>
    <TypeRow type="all" name="AlphaTab.Settings" />
</TypeTable>

## Example - C#

```csharp
var api = new AlphaTabApi<MyControl>(...);
ShowSettingsDialog(api.Settings);
```

## Example - JavaScript

```js
var api = new alphaTab.AlphaTabApi(document.querySelector('#alphaTab'));
showSettingsModal(api.settings);
```

## Example - jQuery

```js
showSettingsModal($('#alphaTab').alphaTab('settings'));
```
