## Description
This event is fired when the rendering of the whole music sheet is finished from the render engine side. There might be still tasks open for
the display component to visually display the rendered components when this event is notified.

    <ParameterRow platform="all" name="renderResult" type="object">
        The render engine specific result object which contains the rendered music sheet:
        - For `svg` it will contain the raw SVG string, 
        - For `html5` it will contain a HTML5 canvas DOM element
        - For `skia` it will contain a `SkImage`
        - For `gdi` it will contain a `Bitmap`
