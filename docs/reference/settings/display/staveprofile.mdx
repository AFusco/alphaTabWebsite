## Description

AlphaTab has various stave profiles that define which staves will be shown in for the rendered tracks.

<PropertyDescription metadata={metadata} />

## Types

<TypeTable>
    <TypeRow name="AlphaTab.StaveProfile" type="net">
        `Default` -  The profile is auto detected by the track configurations.<br />
        `ScoreTab` -  Standard music notation and guitar tablature are rendered.<br />
        `Score` -  Only standard music notation is rendered.<br />
        `Tab` -  Only guitar tablature is rendered.<br />
        `TabMixed` -  Only guitar tablature is rendered, but also rests and time signatures are not shown. This profile is typically used in multi-track scenarios.<br />
    </TypeRow>
    <TypeRow name="string" type="js">
        `default`<br />
        `scoretab`<br />
        `score`<br />
        `tab`<br />
        `tabmixed`    
    </TypeRow>
    <TypeRow name="int" type="js-html">
        `0` - Default<br />
        `1` - ScoreTab<br />
        `2` - Score<br />
        `3` - Tab<br />
        `4` - TabMixed  
    </TypeRow>
</TypeTable>

## Default Value

`Default`
